# System Dependencies for TBsim Documentation
# These must be installed separately from Python packages

## Required System Dependencies

### Pandoc (Required for Jupyter notebook processing)
- **Ubuntu/Debian**: `sudo apt-get install pandoc`
- **macOS**: `brew install pandoc`
- **Windows**: Download from https://pandoc.org/installing.html
- **Minimal Version**: 2.0 or higher

### Python 3.11+ (Required for TBsim)
- **Ubuntu/Debian**: `sudo apt-get install python3.11 python3.11-pip`
- **macOS**: `brew install python@3.11`
- **Windows**: Download from https://www.python.org/downloads/

## Installation Commands by Platform

### Ubuntu/Debian
```bash
sudo apt-get update
sudo apt-get install -y pandoc python3.11 python3.11-pip
```

### macOS
```bash
brew install pandoc python@3.11
```

### Windows (PowerShell)
```powershell
# Install Chocolatey first, then:
choco install pandoc python311
```

## Verification
After installation, verify dependencies:
```bash
pandoc --version
python3.11 --version
```

## Notes
- Pandoc is required for `nbsphinx` extension to process Jupyter notebooks
- Python 3.11+ is required for TBsim compatibility
- These dependencies are automatically installed in GitHub Actions via the workflow
