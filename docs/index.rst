Welcome to TBsim Documentation
=============================

**TBsim** is a comprehensive tuberculosis modeling framework built on the Starsim platform, designed for simulating TB dynamics, interventions, and comorbidities in population-based models.

.. warning::
   TBsim is still in the early stages of development. It is being shared solely for transparency and to facilitate collaborative development. It is not ready to be used for real research or policy questions.
