# Interventions

This folder contains all TBsim intervention modules.

- `bcg.py` -- BCG vaccination product and routine delivery
- `tpt.py` -- TB preventive therapy products and delivery
- `tb_treatment.py` -- TB treatment interventions
- `tb_diagnostic.py` -- TB diagnostic interventions
- `tb_health_seeking.py` -- Health-seeking behavior
- `immigration.py` -- Immigration interventions
- `beta.py` -- Time-varying transmission modifier
- `tb_drug_types.py` -- TB drug type definitions
- `interventions.py` -- Base intervention classes
