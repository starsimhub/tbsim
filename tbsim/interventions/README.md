# Interventions

This folder contains all TBsim intervention modules.

- `dx_products.py` -- TB diagnostic products (`Dx`, `xpert`, `oral_swab`, `fujilam`, `cad_cxr`)
- `tx_products.py` -- TB treatment products (`Tx`, `dots`, `dots_improved`, `first_line`, `second_line`)
- `dx_delivery.py` -- Diagnostic delivery intervention (`DxDelivery`)
- `tx_delivery.py` -- Treatment delivery intervention (`TxDelivery`)
- `bcg.py` -- BCG vaccination product and routine delivery
- `tpt.py` -- TB preventive therapy products and delivery
- `health_seeking.py` -- Health-seeking behavior
- `beta.py` -- Time-varying transmission modifier
- `drug_types.py` -- TB drug type definitions
- `interventions.py` -- Base intervention classes
